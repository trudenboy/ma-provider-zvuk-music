# Провайдер Звук Музыки для Music Assistant

[English](README.md) | Русский

> Стриминг музыки из Звук с поддержкой поиска, просмотра и воспроизведения.

## Возможности

| Функция | Описание |
|---------|----------|
| Обзор | Мне нравится, Исполнители, Альбомы, Плейлисты |
| Поиск | Треки, исполнители, альбомы, плейлисты |
| Воспроизведение | MP3 (320 кбит/с), FLAC (без потерь) |
| Синхронизация библиотеки | «Мне нравится» → библиотека MA |

## Совместимость

| Версия провайдера | Мин. версия MA | Протестировано с MA |
|-------------------|----------------|---------------------|
| 1.x.x             | 2.7.0          | 2.7.0b1             |

## Установка

Устанавливается автоматически как часть Music Assistant.

### Настройка

1. Настройки → Музыкальные источники → Добавить источник → Звук Музыка
2. Введите X-Auth-Token Звука
3. Выберите качество стриминга
4. Сохраните

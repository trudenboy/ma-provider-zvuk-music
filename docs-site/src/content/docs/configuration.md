title: Настройка
description: Получение токена и параметры провайдера Zvuk Music
---

Все настройки доступны через **Настройки → Источники музыки → Zvuk Music**.

## Параметры

| Параметр | Тип | По умолчанию | Описание |
|----------|-----|--------------|----------|
| `token` | Защищённая строка | — | X-Auth-Token аккаунта Zvuk Music. Обязателен для авторизации. См. ниже, как получить. |
| `quality` | Строка (enum) | `high` | Предпочтительное качество звука. См. [варианты качества](#варианты-качества) ниже. |

### Варианты качества

| Значение | Название | Формат | Битрейт |
|----------|----------|--------|---------|
| `high` | Высокое | MP3 | 320 kbps |
| `lossless` | Без потерь | FLAC | Lossless |

## Как получить токен

Провайдер использует X-Auth-Token аккаунта Zvuk Music. Инструкция по его получению находится на странице официальной документации:

[music-assistant.io/music-providers/zvuk/](https://music-assistant.io/music-providers/zvuk/)

## Действия

| Действие | Описание |
|----------|----------|
| Сбросить авторизацию | Удаляет сохранённый токен, позволяя ввести новые учётные данные. |

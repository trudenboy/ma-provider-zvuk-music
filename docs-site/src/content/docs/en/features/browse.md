---
title: Browse
description: Hierarchical navigation through Zvuk Music content in Music Assistant
---

The **Browse** section in Music Assistant lets you freely navigate the provider's catalog without library sync. For Zvuk Music, Browse exposes personalized and editorial playlists.

## Tree structure

```
Zvuk Music
├── Плейлисты для вас (For You)   ← Zvuk synthesis playlists (personalized)
│   └── [playlists…]
└── Подборки (Editorial Picks)    ← editorial playlists (/api3/playlist/)
    └── [playlists…]
```

## Sections

### For You (Плейлисты для вас)

**API:** fixed set of Zvuk synthesis playlist IDs

- Generated by Zvuk's algorithm based on your listening history
- Personal to each account
- Updated automatically by Zvuk — MA reflects the current state
- Folder is hidden if no playlists are available

### Editorial Picks (Подборки)

**API:** `/api3/playlist/?ids=...` — editorial playlists

- Curated by the Zvuk Music editorial team by genre and theme
- Up to 50 current playlists are shown
- The same for all users
- Folder is hidden if playlists are unavailable

## How to open Browse

1. Open Music Assistant
2. Go to the **Browse** section
3. Select the **Zvuk Music** provider
4. Choose a folder — «Плейлисты для вас» or «Подборки»

## Notes

- Browse requires no additional configuration — it works as long as you have a valid token
- Personalized playlists depend on your activity in the Zvuk Music app
- Editorial picks are updated by the Zvuk editorial team independently of MA
- Playlists from Browse can be added to the MA library or played directly

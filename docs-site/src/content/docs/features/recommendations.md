Звук предоставляет два вида рекомендованных плейлистов на главной странице Music Assistant.

## Как это работает

Рекомендации появляются на **главном экране MA** в разделе Discover. Провайдер формирует две группы:

### Плейлисты для вас

**API:** синтез-плейлисты Звука (фиксированный набор ID)

- Формируются алгоритмически на основе вашей истории прослушивания
- Обновляются Звуком автоматически — MA отражает актуальное состояние
- Показываются только при наличии доступных плейлистов

### Подборки

**API:** редакционные плейлисты (`/api3/playlist/?ids=...`)

- Составляются редакцией Звука по жанрам и тематикам
- Актуальный срез до 50 редакционных плейлистов
- Показываются только при наличии доступных плейлистов

## Где отображаются рекомендации

| Место | Описание |
|-------|----------|
| Главная (Discover) | Две группы плейлистов на главном экране MA |

## Примечания

- Рекомендации зависят от вашей истории прослушивания в Zvuk Music
- Редакционные подборки одинаковы для всех пользователей
- Если обе группы пусты, раздел рекомендаций не отображается

---
title: Browse (Обзор)
description: Иерархическая навигация по контенту Zvuk Music в Music Assistant
---

Раздел **Browse** в Music Assistant позволяет свободно перемещаться по каталогу провайдера без синхронизации библиотеки. Для Zvuk Music Browse открывает доступ к персональным и редакционным плейлистам.

## Структура дерева

```
Звук Music
├── Плейлисты для вас   ← синтез-плейлисты Звука (персональные)
│   └── [плейлисты…]
└── Подборки            ← редакционные плейлисты (/api3/playlist/)
    └── [плейлисты…]
```

## Разделы

### Плейлисты для вас

**API:** фиксированный набор синтез-плейлистов Звука

- Формируются алгоритмом Звука на основе вашей истории прослушивания
- Персональны для каждого аккаунта
- Обновляются Звуком автоматически — MA отражает актуальное состояние
- Папка не отображается, если ни один плейлист недоступен

### Подборки

**API:** `/api3/playlist/?ids=...` — редакционные плейлисты

- Составляются редакцией Zvuk Music по жанрам и тематикам
- Отображается до 50 актуальных плейлистов
- Одинаковы для всех пользователей
- Папка не отображается, если плейлисты недоступны

## Как открыть Browse

1. Откройте Music Assistant
2. Перейдите в раздел **Browse** (Обзор)
3. Выберите провайдер **Zvuk Music**
4. Выберите папку — «Плейлисты для вас» или «Подборки»

## Примечания

- Browse не требует дополнительных настроек — он работает при наличии токена
- Персональные плейлисты зависят от вашей активности в приложении Zvuk Music
- Редакционные подборки обновляются редакцией Звука независимо от MA
- Плейлисты из Browse можно добавлять в библиотеку MA или воспроизводить напрямую

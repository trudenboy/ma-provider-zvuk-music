Провайдер поддерживает полнотекстовый поиск по каталогу Zvuk Music.

## Что ищется

| Тип | Поддержка |
|-----|:---------:|
| Треки | ✅ |
| Альбомы | ✅ |
| Исполнители | ✅ |
| Плейлисты | ✅ |

## Как работает

1. MA передаёт поисковый запрос и типы медиа провайдеру
2. Для каждого запрошенного типа выполняется отдельный запрос к API Zvuk
3. Результаты парсятся и возвращаются в MA
4. Количество результатов: до 5 на каждый тип (лимит по умолчанию)

## Особенности

- Поиск выполняется по каталогу Zvuk — не только по личной библиотеке
- Запросы выполняются параллельно для каждого типа медиа
- Если MA не указал тип — выполняется поиск по всем поддерживаемым типам
